# Erazor AI

Professional AI-powered image processing platform for background removal and image upscaling.

## Features

- **Background Removal** - Remove backgrounds from images instantly with AI
- **Image Upscaling** - Enhance image resolution up to 4x with AI
- **Credit System** - Flexible credit-based usage with free tier
- **User Dashboard** - Track usage, manage subscriptions, view history
- **Public Tools** - Try features without signup (limited credits)

## Tech Stack

- Next.js 15 (App Router)
- Supabase
- Tailwind CSS v4
- shadcn/ui
